# 프로젝트 개요 (Project Overview)

## 1. 시스템 소개

**Momentum ETF**는 ETF의 가격 추세를 분석하여 **"상대적으로 가장 우수한 상승 에너지를 가진 종목을 선별"**하는 **반(半)자동 포트폴리오 운용 시스템**입니다.

이 시스템은 사용자가 직접 매일 시장을 모니터링하지 않아도, 데이터에 기반하여 상승 종목의 유지, 교체, 매도 시점을 자동으로 제시함으로써 투자 효율성과 안정성을 동시에 제공하는 것을 목표로 합니다.

## 2. 핵심 철학 및 가치

### "추세에 순응하되, 위험은 철저히 관리한다"

1.  **데이터 기반 의사결정**: 감정이나 직관이 아닌, 수치화된 모멘텀 점수와 알고리즘에 따라 판단합니다.
2.  **다층적 리스크 관리**: 단순한 추세 추종을 넘어, 5버켓 분산 투자를 통해 포트폴리오의 안정성을 확보합니다.
3.  **투명성**: 추천의 이유(점수, 상태)와 백테스트 결과를 명확히 제시하여 사용자가 납득할 수 있는 투자를 돕습니다.

## 3. 주요 기능

*   **📊 통합 추천 시스템 (Unified Recommendation)**: 백테스트 엔진을 기반으로 현재 계좌 상태를 시뮬레이션하여 최적의 매수/매도/교체 신호를 생성하고 Slack으로 알림을 보냅니다.
*   **🛡️ 데이터 안정성 (Robust Caching)**: **Apache Parquet** 포맷을 도입하여 Numpy 버전 충돌 없는 안정적인 가격 데이터 캐싱을 구현했습니다.
*   **💻 직관적인 대시보드 (Modern UI)**:
    *   **요약(Summary) 탭**: 총자산, 원금, 순익, 현금 비중 등 핵심 지표를 한눈에 파악.
    *   **상세(Details) 탭**: 개별 종목의 수익률, 버킷별 비중, 매매 신호를 심층 분석.
    *   **계좌 관리**: 원금 및 현금 잔고를 실시간으로 수정하여 정확한 수익률 계산 지원.

## 4. 시스템 구조 요약

이 프로젝트는 크게 네 가지 파트가 긴밀하게 통합되어 구성됩니다:

1.  **백테스트 엔진 (Backtest)**: 핵심 시뮬레이션 엔진으로, 과거 데이터를 사용하여 전략의 성과(CAGR, MDD)를 시뮬레이션합니다.
2.  **추천 엔진 (Recommend)**: 백테스트 엔진을 기반으로 **가장 최신 시점(오늘)의 시뮬레이션 결과**를 스냅샷으로 추출하여 매매 가이드를 제공합니다.
3.  **튜닝 엔진 (Tune)**: 백테스트를 반복 수행하여 시장 상황에 맞는 최적의 파라미터를 탐색합니다.
4.  **자동화 (Automation)**: GitHub Actions를 통해 매일 정해진 시간에 데이터 수집, 추천, 알림 발송을 수행합니다.

## 5. 사용 시 유의사항

*   본 시스템은 투자를 돕는 **보조 도구**이며, 최종 투자 결정에 대한 책임은 사용자에게 있습니다.
*   과거의 성과(백테스트 결과)가 미래의 수익을 보장하지 않습니다.
*   시장 급변 시 시스템의 신호와 실제 시장 상황 간에 시차가 발생할 수 있습니다.
